# 两数相加

给定两个链表逆序，表示两个非负数，实现相加。

重点：尾插法

```
 if(head==null){
                head=tail=new ListNode((n1+n2+carry)%10);
            }else{
                tail.next=new ListNode((n1+n2+carry)%10);
                tail = tail.next;
            }
```

​		循环截止条件：

```
while(l1!=null||l2!=null)
```

错误思想记录：

```
 if(head==null){
                head=new ListNode((n1+n2+carry)%10);
                head.next=tail
            }else{
                tail=new ListNode((n1+n2+carry)%10);
                tail = tail.next;
            }
```

想当然以为head指向了tail，tail就是head的尾节点了，实际上此时tail指向null并非节点，相当于head指向了null。
