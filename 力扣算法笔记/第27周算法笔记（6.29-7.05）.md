## 一、两数相加

给定两个链表逆序，表示两个非负数，实现相加。

**重点：尾插法**

```
 if(head==null){
                head=tail=new ListNode((n1+n2+carry)%10);
            }else{
                tail.next=new ListNode((n1+n2+carry)%10);
                tail = tail.next;
            }
```

循环截止条件：

```
while(l1!=null||l2!=null)
```

**错误思想记录：**

```
 if(head==null){
                head=new ListNode((n1+n2+carry)%10);
                head.next=tail
            }else{
                tail=new ListNode((n1+n2+carry)%10);
                tail = tail.next;
            }
```

想当然以为 head 指向了 tail，tail 就是 head 的尾节点了，实际上此时 tail 指向 null 并非节点，相当于 head 指向了 null。这里head相当于指向自己了

## 三、翻转链表(简单题)

pre指向前面的节点；

next保存cur的下一个位置；

cur来控制索引连接pre所指的节点；

```
class Solution {
    public ListNode rever(ListNode head) {
        ListNode pre=null;
        ListNode cur=head;
        while(cur!=null){
            ListNode next = cur.next;
            cur.next=pre;
            pre=cur;
            cur=next;
        }
        return pre;//最后pre变成了头结点
    }
}
```

## 四、K个一维链表翻转（困难）

![78298316600](D:\a-笔记\力扣算法笔记\图片\1782983166005.png)

```java
/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode() {}
 *     ListNode(int val) { this.val = val; }
 *     ListNode(int val, ListNode next) { this.val = val; this.next = next; }
 * }
 */
class Solution {
    public ListNode reverseKGroup(ListNode head, int k) {
        ListNode dummy=new ListNode(0,head);
        ListNode pre=dummy;
        while(pre!=null){
            ListNode tail=pre;
            for(int i=0;i<k;i++){
                tail=tail.next;//tail移动k步
                if(tail==null){
                    return dummy.next;//主要退出点
                }
            }
            ListNode next=tail.next;
            myReverse(head,tail);
            ListNode temp=head;
            head=tail;
            tail=temp;//翻转头尾
            
            pre.next=head;
            tail.next=next;
            pre=tail;
            head=pre.next;
        }
        return dummy.next;
    }
    public static void myReverse(ListNode head,ListNode tail){
        ListNode cur=head;
        ListNode pre=tail.next;
        while(pre!= tail){
            ListNode next = cur.next;
            cur.next=pre;
            pre=cur;
            cur=next;
        }
    }
}
```

使用 4 个索引：`pre`、`head`、`tail`、`next`

- `pre` 指向子链表的前一个节点
- `head`、`tail` 表示子链表的头尾节点
- `next` 指向子链表的下一个节点

处理过程：首先子链表自行翻转，再连接子链表。

## 五、随机链表的复制

深拷贝：新链表中存储的都是新的节点，索引也指向新的节点

```java
class Solution {
    public Node copyRandomList(Node head) {
        if(head==null) return null;

        Node cur = head;
        Map<Node,Node> map = new HashMap<Node,Node>();

        while(cur!=null){
            map.put(cur,new Node(cur.val));
            cur=cur.next;
        }
        cur=head;
        while(cur!=null){
            Node copy = map.get(cur);
            copy.next= map.get(cur.next);
            copy.random = map.get(cur.random);
            cur= cur.next;
        }
        return map.get(head);
    }
}
```

建立哈希表存储新节点，重新遍历一遍建立指针索引。
